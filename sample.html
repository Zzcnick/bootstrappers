Columns

Columns are created based on a grid system; there are 12 grid slots. To specify the amount of space a column will take up, add the col-xs-# class to a div to create columns. You may replace "xs" with other sizes among "sm", "md" and "lg" to make dynamic columns that change depending on the screen size. # defines the number of grid spots that a column will take up.

Containers are the building blocks of websites using Bootstrap! They allow us to utilize the 12 unit grid system and properly wrap website elements. The default class is .container and will create a fixed width container. To create a container that will span the length of the viewport, use .container-fluid.

Buttons

You can create buttons using several of Bootstrap's default themes, from "Danger" buttons to Links. All buttons have the .btn, and may contain other classes to define a theme, such as .btn-danger.

Buttons can be grouped together by wrapping them in a div with the .btn-group class. In order to create buttons that span the length of a page, add .btn-block.

Note that adding these in a block will REMOVE the spanning effect!
